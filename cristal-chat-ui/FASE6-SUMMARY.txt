FLUXO DE DADOS:
--------------

1. User digita mensagem → Composer.onSendMessage
2. App.handleSendMessage → useSendMessage.mutate()
3. useSendMessage:
   - onMutate: adiciona mensagem user (chatStore.addMessage)
   - setLoading(true)
   - apiClient.sendMessage() → POST /api/chat
   - onSuccess: adiciona resposta assistant + citations
   - setLoading(false)
4. ChatArea renderiza messages + LoadingDots (se loading)
5. Auto-scroll para última mensagem

PRÓXIMOS PASSOS:
---------------

Para iniciar o projeto:

1. Criar arquivo .env (copiar de .env.example)
2. Garantir backend rodando em http://localhost:8080
3. npm install
4. npm run dev

Backend deve ter endpoints:
- POST /api/chat (body: { message: string })
  Response: { response: string, citations?: Citation[] }
- GET /api/health
  Response: { status: string }
